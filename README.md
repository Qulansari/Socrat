# Socrat – Educational Management System

## Overview

Socrat is a desktop educational management system developed in Java as part of an Object-Oriented Programming course project.
The application provides a simple platform for managing academic data such as schedules, grades, and user profiles while supporting multiple user roles.

The project was developed by a team of three students between September 2023 and December 2023.

## Features

* Secure user authentication with password hashing
* Role-based access control for different user types:

  * Students
  * Teachers
  * Administrators
* Management of academic schedules
* Grade management system
* User profile management
* Relational database integration

## Technologies Used

* **Java**
* **JavaFX** for the graphical user interface
* **SQL** for database management

## System Architecture

The application follows object-oriented design principles and separates core components such as:

* User authentication
* Role-based authorization
* Data management through a relational database
* Graphical user interface built with JavaFX

## Database

The system uses a relational SQL database to store and manage application data, including:

* Academic schedules
* Grading information
* User accounts and profiles

## Project Context

This project was developed as part of an **Object-Oriented Programming course** and focused on applying core software engineering concepts such as:

* object-oriented design
* database integration
* secure authentication
* multi-user system architecture
